Навчання
Створення веб-сайтів з використанням HTML5.2/CSS3 та JavaScript редакції ES2018. Семантичний код у HTML5, структуровані і коректно оформлені сторінки, які проходять HTML та CSS валідатори. Селектори у CSS, а також responsive верстка сторінки, яка підлаштовується під різні пристрої з використанням найбільш трендових технологій – CSS FlexBox та CSS Grid Layout. Замикання, прототипне наслідування та написання асинхронного коду у JavaScript. Робота з DOM-моделлю сторінки, реагувати на події, генерувати динамічний контент, а також взаємодіяти з бекендом за допомогою технологій Ajax та Fetch.
